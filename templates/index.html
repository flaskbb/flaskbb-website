Open Source

FlaskBB is open source and the whole source code is available on GitHub. It is being developed under the BSD 3-Clause license.

Plugin System

Due to the plugin system it is possible to extend FlaskBB without touching a single line of code. You just drop the plugin into the plugins directory and activate it in the managment panel.

Themes

Are you sick of the standard theme or you simply want to create a customized style for your site? No problem! You can customize every template in FlaskBB. Besides that, you can also install community made themes.

Modern & Classic

Our goal is to develop a forum system with modern technologies and sticking to already well tested and approved layouts.
