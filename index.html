FlaskBB a lightweight forum software

FlaskBB is forum software written in Python using the microframework Flask. You can easily create new topics, posts and send other users private messages. It also includes basic administration and moderation tools.

Download »

Features

FlaskBB comes with a lot of inbuilt features, such as private messages, a topic tracker, a unread/read tracker, a own permissions system (!) and last but not least with a simple admin interface.

Easy to use

It is built for humans from humans, so it should be really easy to use. We try to keep the things simple and clear.

Built with Flask

Flask is a microframework for Python based on Werkzeug, Jinja 2 and good intentions.

Administrator interface. It'll blow your mind.

We try to keep the administrator interface as clean and simple as possible. Nobody should have to learn how to administrate a forum.

Permission system. See for yourself if you can.

Another nice feature that we provide, is a permission system. For example, I you want to have a special group with special permissions - just create one - it is possible!

Moderation tools. Easy accessible

Do you ever wanted to easily moderate a forum? No problem - now you can! Simply click on a topic (you need to have the right permissions of course) and do what your heart desires.
